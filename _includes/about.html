BST

BST is a language that allows you to create branching stories and small text-based games. It is made to be as simple and straightforward as possible, but still be powerful. But BST isn't only just this : it has a scripting mechnism, sounds, interface customization, and more!

The BST format is open, and documentation is available from the Download section of the website. However, by itself, your text file won't do anything. Much like an image file, you'll need some program to open it; and this is exactly what OpenBST does!

OpenBST

OpenBST is a simple and cross-platform program made to read and interact with any BST formatted file. It aims to be up to date with the latest new fancy features.

Its interface is simple enough for newcomers while containing everything needed for power users and story creators.

Start creating your own stories today : it's 100% free! :)
